#### 变量与常量声明

1.基础变量<br />PHP 中的变量用一个美元符号后面跟变量名来表示。变量名是区分大小写的。<br />变量名与 PHP 中其它的标签一样遵循相同的规则。一个有效的变量名由字母或者下划线开头，后面跟上任意数量的字母，数字，或者下划线。按照正常的正则表达式，它将被表述为：`[a-zA-Z\x7f-\xff][a-zA-Z0-9\x7f-\xff]*` 。

```
<?php

$username = "Leon Kennedy";         // 通过
$_username = "Leon Kennedy";        // 通过
$username1 = "a1";                  // 通过
$user.name = "Leon Kennedy";        // 报错
$用户名称 = "初七";                   // 通过
$UserName = "Leon Kennedy";         // 通过
$userName = "Leon";                 // 通过
$user_name = "Cissy";               // 通过
$1 = 100;                           // 报错

```
